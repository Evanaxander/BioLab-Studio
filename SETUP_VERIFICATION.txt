QUICK START INSTRUCTIONS
================================================================================

Step 1: Open Terminal in VS Code
  - Press Ctrl+` (Windows/Linux) or Cmd+` (Mac)
  - Or use Terminal menu → New Terminal

Step 2: Navigate to Project
  Command: cd biolab-studio-pro

Step 3: Install Dependencies
  Command: npm install
  Duration: 30-60 seconds

Step 4: Start Server
  Command: npm start
  Output: Server running at http://localhost:3000

Step 5: Open in Browser
  URL: http://localhost:3000
  Application loads immediately

SUPPORT AND NEXT STEPS

For setup assistance:
  1. Verify Node.js installation: node --version
  2. Verify npm installation: npm --version
  3. Check available disk space for npm install
  4. Ensure port 3000 is not in use

  Add new modules by extending the models object
  Modify difficulty content in the difficulty adaptation sections

For deployment:
  This project is production-ready and can be deployed to any Node.js hosting
  Copy the entire directory to your target environment
  Run npm install and npm start on the target system
  Application will run on http://localhost:3000
